------------------------------------
Convert from DM42n to C47
------------------------------------

In DM42n, go to [f] SETUP
6. Activate USB Disk      | Note: Backup contents of original DM42n FAT disk to computer

5. System
2. Enter System Menu
4. Reset to DMCP menu
5. Format FAT Disk        | Note: Format not required, but does help for clean install 
EXIT                      | ..
4. Reset to DMCP menu
6. Activate USB Disk      | Note: Observe the disk name is now DM42

Copy the C47 files to the DM42n drive. Decide what you need on your drive: Options a, b & c below:
a. C47.pg5 and testPgms
b. All files in c47-dmcp5
c. Only the necessary files and directories in c47-dmcp5:
   - Create a directory OFFIMG on the DM42 drive and copy the contents of the offimg folder to it
   - Copy PROGRAMS to DM42
   - Copy testPgms.bin from resources to DM42 (if you want the test programs)
   - Copy C47.pg5  to DM42
   - C47_qspi.bin is not needed and needs not be copied. It is there for completeness.

*** finish up, i.e. release/unmount the DM42 flash drive from Mac/PC

EXIT
ENTER
3. Load Program
Choose C47.pg5
ENTER
ENTER
ENTER
EXIT

C47 is loaded


If C47 is not loaded:
Should the DMCP report a QSPI error, copy the file DM42_qspi_3.x.bin to the flash drive
Unmount the flash drive
4. Load QSPI
It may reboot. You may need to press the pinhole switch on the back of the calculator while holding F1.
2. Load Program, select C47.pg5
3. Run Program.

If still not loaded, follow the procedure below to load DM42n, and then continue this procedure from step 1.


---------------------------
Return from C47 to DM42n
---------------------------

ON
[f] MODE
[f] DMCP
6 Activate USB Disk
Download DMCP5_flash_3.50t40_DM42-3.23.bin from SwissMicros web site, Technical, DM42.
Copy DMCP5_flash_3.50t40_DM42-3.23.bin to DM42n's flash USB disk
Unmount flash Disk

EXIT
ENTER
ENTER
EXIT

DM42 is loaded
